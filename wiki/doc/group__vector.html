Vector

Synopsis

The vector structure and accompanying functions loosely mimic the STL C++ vector class.

It is used by Texture atlas (for storing nodes), Texture font (for storing glyphs) and Font manager (for storing fonts). More information at http://www.cppreference.com/wiki/container/vector/start

Example Usage: 

 #include "vector.h"

 int main( int arrgc, char *argv[] )
 {
   int i,j = 1;
   vector_t * vector = vector_new( sizeof(int) );
   vector_push_back( &i );
 
   j = * (int *) vector_get( vector, 0 );
   vector_delete( vector);

   return 0;
 }
