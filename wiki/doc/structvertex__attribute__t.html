Field Documentation

GLint vertex_attribute_t::size

Number of components per generic vertex attribute.

Must be 1, 2, 3, or 4. The initial value is 4.

GLsizei vertex_attribute_t::stride

byte offset between consecutive generic vertex attributes.

If stride is 0, the generic vertex attributes are understood to be tightly packed in the array. The initial value is 0.

GLenum vertex_attribute_t::type

data type of each component in the array.

Symbolic constants GL_BYTE, GL_UNSIGNED_BYTE, GL_SHORT, GL_UNSIGNED_SHORT, GL_INT, GL_UNSIGNED_INT, GL_FLOAT, or GL_DOUBLE are accepted. The initial value is GL_FLOAT.
