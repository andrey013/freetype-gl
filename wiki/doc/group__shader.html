Shader

Synopsis

Simple functions for loading/building a shader from sources.

Example Usage: 

 #include "shader.h"

 int main( int arrgc, char *argv[] )
 {
     GLuint shader = shader_load("shader.vert", "shader.frag");

     return 0;
 }
